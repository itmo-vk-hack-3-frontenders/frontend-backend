# ITMO / VK / OK Hack

## Запуск проекта

Проект использует Makefile, для запуска проекта необходимо выполнить команду:

```bash 
make init
```

| Адрес                          | Описание |
|--------------------------------|----------|
| http://localhost:1333          | Frontend |
| http://localhost:2333          | Backend  |
| http://localhost:2333/api/docs | Swagger  |

### Список доступных команд

```bash
make
```
